# Expensify

The Expensify App

## Getting Started

For development run "yarn dev-server" to launch a browser and enable hot-reloading

### Prerequisites

Run "yarn" to install the required dependencies.

## Running the tests

Run "yarn test"

## Deployment

Run "yarn build" and it will build all dependencies into the public folder
